# 📅 Andy Knows! - Landing Page

> **Status do Projeto:** Concluído ✅

O **Andy Knows!** é uma landing page moderna e minimalista para uma agenda virtual. Este projeto foi criado para demonstrar a minha capacidade de transformar protótipos de alta fidelidade (Figma) em interfaces reais, utilizando as melhores práticas de **HTML5** e **CSS3**.



---

## 🎨 O Desafio (Design)
O foco principal foi o **Pixel Perfection**. Como venho da área de design, busquei transpor cada detalhe do protótipo:
* **Tipografia:** Uso da fonte *Inter* para garantir legibilidade e modernidade.
* **Profundidade:** Aplicação de sombras em camadas (`box-shadow`) para criar um efeito de interface flutuante (Glassmorphism leve).
* **Cores:** Paleta em tons pastéis (`#E3F4FF`) para reduzir a fadiga visual, com contrastes estratégicos nos botões de ação.

## 🛠️ Tecnologias e Conceitos
Para construir este projeto, utilizei:
* **HTML5 Semântico:** Estruturação focada em acessibilidade e SEO (uso de `<header>`, `<main>`, `<section>`).
* **CSS Flexbox:** Domínio de eixos (horizontal e vertical) para criar um layout responsivo e organizado.
* **UX/UI:** Micro-interações com estados de `:hover` nos botões para melhor feedback ao usuário.
* **Mobile-First Thinking:** Planejamento de como os elementos se empilham em telas menores.
